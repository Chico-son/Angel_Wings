## Overview

Angel_Wings is a solo summer project focused on building a 3D fighter-jet dogfighting game in Unity and get some C# under my belt.

The goal is not to create a military simulator, it's a fun summer project as a rising senior.

I love working with game engines and air-combat games.

The game is inspired by War Thunder.

## Planned Features

### Flight System           
- Pitch
- Roll
- Yaw
- Throttle control
- Third-person camera 

### Combat
- Machine guns
- Damage system
- Enemy targeting
- Missile lock-on
- Homing missiles

### Enemy AI
- Patrol behavior
- Chase behavior
- Attack behavior
- Basic evasive maneuvers

### User Interface
- Crosshair
- Speed indicator
- Altitude indicator
- Missile lock indicator
- Enemy markers

### Gameplay
- Mission objectives
- Enemy waves
- Victory and defeat conditions
- Pause menu
